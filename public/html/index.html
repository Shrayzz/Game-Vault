Project Overview

This project aims to develop a Personal Library Management Application tailored specifically for video game collections.
The application will allow users to catalog and organize their video games efficiently, with advanced search and filtering options.
Users will be able to track not only what games they own but also where they are stored (e.g., shelf, box, storage unit).

Storage Features

  • Search bar
  • Sort by tags / game categories
  • Create custom lists and add games to them
  • Mark games as favorites
  • Game photo/cover (Optional)
  • View game details (associated tags, release date, publisher/developer, required configurations, etc.)
  • Register/login system, Steam account linking
  • Add/remove friends
  • Friend list, messaging between friends
  • Comment system for games
  • Game achievements
  • Game recommendations
